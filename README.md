
## API de Produtos 
*Operações básicas:*

- Listar todos os produtos
   
- Listar um produto (pelo ID)
- Criar um produto
- Editar um produto
- Excluir um produto

### Tecnologias ultilzadas
| Tecnologia                | Descrição                                                                           |            
| ------------------------- | ----------------------------------------------------------------------------------- | 
| `H2`                      | Banco de dados local(teste)                                                         | 
| `jpa`                     | Persistência de dados.                                                              | 
| `SpringBoot`              | configuração e publicação de aplicações                                             |
